This REST API for Home Silk using Node, Express, Typescript & MongoDB.

### Prerequisites

**Node version 14.x**

### Install packages

```shell
npm i
```

### Setup MongoDB URL

In `.env`:

```js
 DB_URL = ''; // DB URI
```

### Start the app

```shell
npm start
```

## Available commands

Running commands with npm `npm run [command]`

| command         | description                              |
| :-------------- | :--------------------------------------- |
| `start`         | Starts a development instance of the app |
